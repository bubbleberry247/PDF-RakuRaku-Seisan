# docs/runbooks/recoru/ - レコル関連Runbook

## このファイルの目的
レコル関連シナリオの運用手順（Runbook）の目次・ナビゲーション

## 対象読者
- 運用担当者（日次/週次/月次の運用）
- 障害対応者（トラブルシューティング）
- 新規参画者（オンボーディング）

## 更新ルール
- **Runbook**: シナリオ単位で作成
- **業務ルール**: 業務ドメイン知識は [docs/domain/recoru/](../../domain/recoru/) を参照
- **追記**: 新シナリオのRunbookは本READMEの一覧に追加

---

## 重要：RunbookとDomainの分離

### 書く場所の違い

| 内容 | 書く場所 | 例 |
|------|---------|-----|
| **業務ルール・用語定義** | `docs/domain/recoru/` | 勤務区分の定義、振休と代休の違い |
| **運用手順・操作手順** | `docs/runbooks/recoru/` | シナリオの起動方法、障害対応手順 |
| **画像・帳票見本** | `docs/assets/recoru/` | スクリーンショット、帳票サンプル |

### 相互参照

- Runbookから業務ルールを参照する場合 → `[用語の定義](../../domain/recoru/00_overview.md)` 形式でリンク
- 業務ルールはdomainに1箇所だけ書き、Runbookでは参照のみ（コピー禁止）

## Runbookファイル命名規則

```
runbook_s{シナリオ番号}_{機能名}.md
```

### 例
- `runbook_s37_leave_registration.md` - S37 レコル公休・有休登録
- `runbook_s43_paid_leave.md` - S43 TBD
- `runbook_s47_attendance_check.md` - S47 出退勤確認
