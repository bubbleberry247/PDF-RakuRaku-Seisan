# doboku-14w image assets

This folder stores static image assets for the `doboku-14w-training` web app.

## Naming convention

- One image per question (recommended).
- File name:
  - `{qId with "-" replaced by "_"}{ext}`
  - Example: `R7gakkaA-001` -> `R7gakkaA_001.png`

## Hosting (GitHub raw)

The app can reference images via GitHub raw URLs:

`https://raw.githubusercontent.com/bubbleberry247/PDF-RakuRaku-Seisan/master/docs/assets/doboku-14w/images/{fileName}`
