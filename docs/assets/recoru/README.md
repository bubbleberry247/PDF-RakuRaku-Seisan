# docs/assets/recoru/ - レコル関連アセット

## このファイルの目的
レコル関連のスクリーンショット・帳票見本の置き場所と命名規則を定義する

## 対象読者
- ドキュメント作成者（画像配置時の参照）
- 運用担当者（見本画像の参照）

## 更新ルール
- **配置**: 画像ファイルはこのディレクトリ配下に配置
- **命名規則**: 下記の命名規則に従う
- **機微情報**: マスキング必須（下記参照）

---

## ディレクトリ構造

```
docs/assets/recoru/
├── README.md           # このファイル
├── screenshots/        # スクリーンショット
│   └── .gitkeep
└── report_samples/     # 帳票見本
    └── .gitkeep
```

---

## ファイル命名規則

### スクリーンショット

```
{yyyymmdd}_{画面名}_{説明}.png
```

#### 例
- `20260117_login_screen.png` - ログイン画面
- `20260117_leave_application_input.png` - 休暇申請入力画面
- `20260117_leave_application_confirm.png` - 休暇申請確認画面
- `20260117_attendance_list_monthly.png` - 勤怠一覧（月次）

### 帳票見本

```
{yyyymmdd}_{帳票名}_{説明}.{拡張子}
```

#### 例
- `20260117_attendance_report_sample.pdf` - 勤怠一覧表サンプル
- `20260117_overtime_report_sample.xlsx` - 残業一覧表サンプル

## 機微情報の取り扱い（重要）

### マスキング必須項目

| 項目 | 対処 |
|------|------|
| 氏名（実名） | モザイク or ダミー名に置換 |
| 社員番号 | モザイク or ダミー番号に置換 |
| メールアドレス | モザイク |
| 電話番号 | モザイク |
| 住所 | モザイク |
| URL（ドメイン部分） | 必要に応じてマスキング |
| 認証情報 | **絶対に含めない** |

### マスキング方法

1. **画像編集ツール**でモザイク処理
2. **ダミーデータ**に置換（テスト環境で撮影）
3. **必要最小限の領域**のみキャプチャ

### 共有禁止ファイル

以下のファイルはこのディレクトリに**配置禁止**：

- 本番環境の生データ（マスキングなし）
- 認証情報を含む画像
- 個人情報を含む帳票（マスキングなし）
