# Plan A（ストレステスト型） 講師スクリプト v1

## 使い方
- スライド番号ごとに、読み上げの骨子（2〜6文）を入れています
- セミナー内では“個別の答え”を出さず、面談で数字に落とす導線を守ってください

## 1. 富裕層の四大ストレスを“構造”で理解する

スライド要点:
- 税・社会保険・教育・老後を、データで腹落ちさせる
- 個別の答えは出しません（答えは面談で“あなたの数字”に落とす）
- Version: 2026-02-07 v1

話すこと（骨子）:
- 今日は“答え”を渡す回ではありません。
- 同じ年収帯でも、家族構成・住宅・教育・年金で正解が変わるからです。
- 今日は“構造と変数”を理解して、あなたの数字に落とすのは無料面談でやります。

## 2. 今日のゴール（言うこと / 言わないこと）

スライド要点:
- 言う: 構造と変数（何が負担を増減させるか）
- 言う: 2つのペルソナで“悩みが変わる”を可視化
- 言わない: 商品推奨・資産配分の正解・税務法務の断定

話すこと（骨子）:
- 最初に約束です。
- 今日は“構造と変数”まで。『あなたはこれを買うべき』はやりません。
- 税務・法務も個別事情が必要なので、一般論までに留めます。
- この後の無料面談でライフプランを作り、そこで初めて“あなたの答え”を出します。

## 3. アジェンダ（60分）

スライド要点:
- 1. データで現実を見る（A/Bの家計）
- 2. 四大ストレスの構造（税・社保・教育・老後）
- 3. Persona A: 住宅×教育×老後の重なり
- 4. Persona B: 単身×老後×資産運用の順序
- 5. 介護・相続（1枚だけ）
- 6. 無料面談の案内（その場で予約）

話すこと（骨子）:
- 流れはこの6つです。
- 前半でデータを出して、後半で2つの代表ケースをストレステストします。
- 最後に介護・相続は1枚だけ触れ、無料面談の予約で締めます。

## 10. 家計は“システム”である

話すこと（骨子）:
- 家計は“気合”で回すものではなく、“システム”として設計します。
- 収入が入力、税と社保が摩擦、支出が出力、資産と負債が状態。
- この全体像が見えると、打ち手の順序が決まります。

## 11. 今日扱う“変数”一覧

スライド要点:
- 税: 税率・控除・家族構成・所得の形
- 社会保険: 標準報酬月額・年齢・扶養
- 教育: いつ・何人・公立私立・塾
- 住宅: 借入額・金利・期間・固定/変動
- 老後: 生活水準・年金・退職金・寿命

話すこと（骨子）:
- 今日扱う変数はこの範囲に絞ります。
- 重要なのは、1個ずつではなく“重なり”で負担が増えること。
- ここから各変数をざっと俯瞰します。

## 12. 税（変数の地図）

スライド要点:
- 増えやすい: 所得・賞与・副収入、扶養の変化
- 減りやすい: 各種控除・非課税枠・制度（詳細は面談）
- ポイント: 税は“率”より“構造”を掴む

話すこと（骨子）:
- 税は“率”を暗記するより、何が増減させるかの構造を掴むのが早いです。
- 控除や制度は個別事情で変わるので、今日は名前だけ。
- 面談であなたの条件に当てはめて、効果の大きい順を出します。

## 13. 社会保険（変数の地図）

スライド要点:
- 増えやすい: 標準報酬月額の上昇、年齢（介護保険等）
- 見落とし: 税より“社保が重い”ケースが普通にある
- ポイント: 手取り設計は“税と社保セット”で

話すこと（骨子）:
- 今日一番の盲点が社会保険です。
- 税より社保が重いケースは普通にあります。
- 手取りを増やす設計は、税と社保をセットで見ないと崩れます。

## 14. 高額療養費（“ゼロ”にはならない）

スライド要点:
- 自己負担の上限はあるが、出費がゼロになるわけではない
- 見落としやすい: 収入減（休職）と同時に起きる
- 備えは“制度の理解”と“余白の確保”

話すこと（骨子）:
- 高額療養費は重要な制度です。
- ただし“ゼロになる”制度ではありません。
- 医療費だけでなく、休職や働き方の変化で収入が下がることがセットで起きます。

## 15. 教育費（タイミングが勝負）

話すこと（骨子）:

## 16. 老後資金: 生活水準は下げにくい（ラチェット効果）

スライド要点:

- 年金だけで今の生活水準を保てるか？が本質

話すこと（骨子）:
