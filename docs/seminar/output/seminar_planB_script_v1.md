# Plan B（タイムリー導入型） 講師スクリプト v1

## 使い方
- スライド番号ごとに、読み上げの骨子（2〜6文）を入れています
- セミナー内では“個別の答え”を出さず、面談で数字に落とす導線を守ってください

- 個別の答えは面談であなたの数字に落とします。

## 2. 今日のゴール（構造と変数）

スライド要点:
- 変数が動いても折れない“家計の耐久性”を理解する
- 2つのペルソナで“設計が変わる”を確認
- 答え（最適解）は面談で提示

話すこと（骨子）:
- ゴールは“構造と変数”です。
- 正解は1つではありません。
- ペルソナが違えば設計が変わる。それを腹落ちさせます。

## 3. アジェンダ（60分）

スライド要点:
- 1. 今起きている変化（枠だけ）
- 2. データで現実を見る（A/B家計）
- 3. 四大ストレスの俯瞰（変数の地図）
- 4. 2ペルソナ比較（A→B）

- 6. 無料面談の案内（予約）

話すこと（骨子）:

- ただし未来予測が目的ではありません。
- 変化があるほど、可視化と順序が重要という話につなげます。

## 4. 今起きている変化（タイムリー枠）

スライド要点:
- 物価・金利: 金利のある世界に戻りつつある
- 税制: 一時的な減税や制度改正が起きる（例: 定額減税）
- 資産形成: 新NISAなど“非課税枠”の重要性が増す
- 社会保険: 子育て支援金など、負担構造が変わり得る

出典/注記: 出典例（要差し替え）: 日本銀行（金融政策/金利）, 国税庁（税制）, 金融庁（NISA）, こども家庭庁（子育て支援）

話すこと（骨子）:
- ここは“当てる”話ではなく、前提が揺れるという話です。
- 金利・税制・制度は変わります。
- 前提が動くほど、早めに自分の家計を見える化して、打ち手を順序で持つのが強い。

## 5. 変化がある時代に必要なこと

スライド要点:
- 変数を当てるより、変数が動いても耐える設計
- 最初の一手は“現状把握（ライフプラン）”
- 今日は構造、答えは面談で

話すこと（骨子）:
- 変数を当てるのは難しい。
- だから設計。変数が動いても耐える設計を持つ。
- その設計図がライフプランで、答えは面談で出します。
