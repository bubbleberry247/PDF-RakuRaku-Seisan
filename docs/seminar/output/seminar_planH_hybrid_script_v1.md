- 個別の最適解は面談で（今日は“構造とテンプレ”まで）
- Version: 2026-02-07 v1

話すこと（骨子）:
- 今日は“当てる話”ではなく、“耐える設計”の話です。
- 年収が高くても、税・社保・教育・老後が重なると余白は想像以上に薄くなります。
- 今日は構造とテンプレまで。
- あなたの数字に落として最適解候補を出すのは、無料の個別面談で行います。

## 12. 今日扱う“変数”一覧（地図）

スライド要点:
- 税: 税率・控除・家族構成・所得の形
- 社保: 標準報酬月額・年齢・扶養
- 教育: いつ・何人・公立私立・塾
- 住宅: 借入額・金利・期間・固定/変動
- 老後: 生活水準・年金・退職金・寿命

話すこと（骨子）:

- 次に制度が動く例を1つだけ見て、すぐテンプレに入ります。

## 13. 税制は動く（改正大綱の例）

スライド要点:
- 控除・非課税枠・住宅税制などが毎年見直される
- 例: 基礎控除等の対応、通勤手当の非課税限度額、住宅ローン控除、NISAの見直し 等
- 結論: 制度を“当てる”より、動いても耐える設計が強い

出典/注記: 出典: 財務省『令和8年度税制改正の大綱』（税制は毎年動く）

話すこと（骨子）:
- 税制は毎年動きます。
- 細かい制度を暗記するより、家計側の耐久性を上げる方が再現性が高い。
- 制度活用は最後に最適化で十分です。

## 14. 高額療養費（“ゼロ”にはならない）

スライド要点:
- 自己負担の上限はあるが、出費がゼロになる制度ではない
- 見落としやすい: 収入減（休職）と同時に起き得る
- 備えは『制度理解 + 余白 + 固定費設計』

出典/注記: 出典: 厚生労働省『高額療養費制度を利用される皆さまへ』

話すこと（骨子）:
- 高額療養費は重要な制度です。
- ただしゼロになる制度ではない。
- 医療費と同時に収入が落ちると、家計の耐久性が試されます。

## 15. 教育費は“金額”より“タイミング”が勝負

出典/注記: 出典: 文部科学省『令和5年度 子供の学習費調査（調査結果の概要）』

話すこと（骨子）:

- 例: 小学校の学習費総額（年間平均） 公立約36.7万円 / 私立約174.2万円（文科省）

## 17. 老後資金: 生活水準は下げにくい（ラチェット効果）

- 年金だけで今の生活を維持できるか。
- 足りないなら、いつから何年分をどう埋めるか。これが逆算です。

## 22. ストレステスト① 金利上昇（参考例）

出典/注記: 参考例（概算）: 5,000万円 / 35年 / 元利均等（ボーナス返済なし）

## 23. ストレステスト② 病気/休職/収入減

話すこと（骨子）:

- ここも面談であなたの条件に落として、弱点を潰します。
