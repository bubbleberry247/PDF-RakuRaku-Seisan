# セミナー構成案（60分）v1

更新日: 2026-02-07

## 1. ゴール（このセミナーで「言う」こと / 「言わない」こと）

### 言う（60分でやること）
- 日本の制度（税・社会保険・高額療養費・教育費・住宅・老後）を「構造」と「変数」で整理する
- 富裕層の四大ストレス（税金 + 社会保険 + 教育資金 + 老後）を、データで腹落ちさせる
- 2つのペルソナで「同じ年収帯でも悩みが変わる」ことを可視化する
- 結論: ベストソリューションは人により違うので、まずライフプランで現状把握が必要
- セミナー終了後に「無料の個別面談」をその場で予約してもらう導線を作る

### 言わない（セミナー内で避けること）
- 「あなたはこれをやるべき」系の個別最適・商品推奨・資産配分の正解
- 税務/法務の断定（個別事情が必要なので個別面談へ）

## 2. 想定ターゲット（前提）

- 20代〜40代
- 世帯年収1000万円以上（職種不問、主に大手企業の会社員）
- 会場: 組合経由の社内オフラインセミナー
- 男性比率: 8割以上を想定（理詰め・データ・グラフ中心が刺さりやすい）

## 3. ペルソナ（このセミナー内の代表ケース）

### Persona A（メイン）
- 35歳 夫婦
- 妻: 専業主婦
- 子ども2人
- 論点: 子育て + 住宅（買う/買わないではなく「家計の耐久性」）

### Persona B（メイン）
- 40歳 単身（離婚後の単身も含む）
- 大企業勤務
- 論点: 将来不安からの資産運用、老後、親の介護/相続は最後に箇条書きで軽く

### 5.3 五分位の境界値（年収レンジの目安）

- 五分位1: 351万円以下
- 五分位2: 510万円以下
- 五分位3: 689万円以下
- 五分位4: 932万円以下
- 五分位5: 上限なし（概ね 932万円超）

「年収1000万円」との距離感（スライド用表現）:
- 年収1000万円は五分位5に含まれる
- ただし五分位5は「1000万以上」ではなく「上位20%」なので幅が広い
- 本セミナーの統計（五分位5）は、1000万層の近似として採用している
