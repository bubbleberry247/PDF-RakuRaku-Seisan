## 2. 用語集

### 2.1 基本用語

| 用語 | 読み | 定義 | 備考 |
|------|------|------|------|
| 勤務区分 | きんむくぶん | 勤務の種類を表すコード。詳細は[10_kbn_dictionary.md](./10_kbn_dictionary.md)参照 | 最重要 |
| 打刻 | だこく | 出勤・退勤時刻の記録 | |
| 申請 | しんせい | 休暇取得等を上長に申し出ること | |
| 承認 | しょうにん | 申請を認めること | |
| 差戻し | さしもどし | 申請を却下し、修正を求めること | |
| 締め | しめ | 月次の勤怠データを確定すること | |
| 締め後修正 | しめごしゅうせい | 締め処理後に勤怠データを修正すること | 要権限 |

### 2.2 勤怠関連用語

| 用語 | 読み | 定義 | 備考 |
|------|------|------|------|
| 所定労働時間 | しょていろうどうじかん | 契約で定められた1日の労働時間 | TBD時間 |
| 法定労働時間 | ほうていろうどうじかん | 労働基準法で定められた上限 | 8時間/日、40時間/週 |
| 残業 | ざんぎょう | 所定労働時間を超えた労働 | |
| 深夜残業 | しんやざんぎょう | 22:00〜5:00の労働 | 割増対象 |
| 休日出勤 | きゅうじつしゅっきん | 休日に出勤すること | 割増対象 |

### 2.3 休暇関連用語

| 用語 | 読み | 定義 | 備考 |
|------|------|------|------|
| 有給休暇 | ゆうきゅうきゅうか | 給与が支払われる休暇 | 年次付与 |
| 公休 | こうきゅう | 会社が定めた休日 | |
| 振替休日 | ふりかえきゅうじつ | 休日出勤の代わりに取得する休日 | |
| 代休 | だいきゅう | 休日出勤後に取得する休日 | 振休との違い注意 |
| 特別休暇 | とくべつきゅうか | 慶弔等の特別な休暇 | |

### 2.4 権限関連用語

| 用語 | 読み | 定義 | 備考 |
|------|------|------|------|
| 一般ユーザー | いっぱんゆーざー | 自分の勤怠のみ参照・申請可能 | |
| 承認者 | しょうにんしゃ | 部下の申請を承認できるユーザー | |
| 管理者 | かんりしゃ | 全員の勤怠を参照・修正できるユーザー | |
| システム管理者 | しすてむかんりしゃ | マスタ設定等が可能なユーザー | |
