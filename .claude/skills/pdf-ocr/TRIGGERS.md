# PDF OCR Skill - トリガー定義

## 自動起動キーワード

このスキルは以下のキーワードを含むユーザー入力で自動的に起動されます。

### 機能・処理名
- PDF OCR
- OCR処理
- テキスト抽出
- 文字認識
- 請求書認識
- 領収書認識

### 技術・ライブラリ
- PyMuPDF
- fitz
- pdf2image
- Tesseract
- OpenCV
- 画像前処理

### 処理名
- 回転補正
- 傾き補正
- deskew
- 影除去
- 画像強調

### データ抽出
- 日付抽出
- 金額抽出
- 取引先抽出
- インボイス番号
- 事業者登録番号

## 除外キーワード

以下のキーワードのみの場合は起動しない（他スキルの領域）:
- RK10シナリオ（→ rk10-scenario skill）
- 楽楽精算アップロード（→ rakuraku skill）

## 使用例

```
ユーザー: PDFからテキストを抽出したい
→ pdf-ocr スキル起動

ユーザー: 請求書PDFの金額を読み取りたい
→ pdf-ocr スキル起動

ユーザー: スキャンPDFの傾きを補正したい
→ pdf-ocr スキル起動

ユーザー: OCRの精度を改善したい
→ pdf-ocr スキル起動
```
