# RK10 Steps（STEP_ID + 行番号）

## 必須列の説明

| 列名 | 説明 | 例 |
|------|------|-----|
| STEP_ID | 固定ID（主キー）、変更禁止 | S010, S020 |
| RK10行番号 | RK10編集用（変動可） | 15, 23 |
| 画面/場所 | 操作対象の画面やシステム | 楽楽精算ログイン画面 |
| 操作(Action) | 何をするか | クリック、入力、待機 |
| 入力/参照 | 使用するデータ | config:LOGIN_ID |
| 成功条件（観測） | 観測可能な状態 | ダッシュボード表示 |
| 待機条件 | 何を待つか | networkidle, 要素出現 |
| 例外分類(error_code) | error_codes.mdとの連携 | AUTH_EXPIRED, TIMEOUT |
| リトライ | Yes/No、回数 | Yes(3) |
| 再実行 | このSTEPから再実行可能か | Yes/No |
| 副作用 | 重複実行時のリスク | 二重登録なし |
| 証跡 | ログ/スクショの種類 | screenshot, log |
| 保存場所 | LOCAL/PRODフルパス | LOCAL:C:\...\logs\ |

## 注意事項

1. **STEP_IDは固定**: コードとの紐付けに使用。変更禁止。
2. **RK10行番号は変動可**: RK10側の編集で行が増減した場合は更新する。
3. **再実行列**: "No"の場合、失敗時は前のSTEPから再実行が必要。
4. **副作用列**: 空欄の場合は「副作用なし（再実行安全）」を意味する。
