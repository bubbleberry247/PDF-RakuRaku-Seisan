# 処理フロー

## 1. 全体フロー

```
[開始]
   │
   ▼
[事前チェック] ──失敗──▶ [エラー通知] ──▶ [終了]
   │
   成功
   │
   ▼
[データ取得]
   │
   ▼
[データ処理]
   │
   ▼
[結果出力]
   │
   ▼
[事後処理]
   │
   ▼
[完了通知]
   │
   ▼
[終了]
```

---

## 3. 状態遷移

| 現在状態 | イベント | 次状態 | アクション |
|----------|----------|--------|------------|
| 待機中 | 起動トリガー | 実行中 | 処理開始 |
| 実行中 | 正常完了 | 完了 | 完了通知 |
| 実行中 | エラー発生 | エラー | エラー通知 |
