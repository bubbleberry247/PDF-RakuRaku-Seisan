# RK10シナリオ実行手順の記録

## 目的
RK10でRKSファイルを自動実行するための正しい手順を記録し、AGENTS.mdのProject Memoryに追記する。

## ユーザーから教わった正しい手順

### RKSファイル実行手順

1. **RKSファイルをダブルクリック**（エクスプローラーから）
   - RK10が起動し、シナリオが読み込まれる

2. **ライセンス種別選択ダイアログ**
   - 「RPA開発版AI付き」が選択された状態で表示される
   - **「OK」ボタンをクリック**

3. **シナリオエディタが開く**
   - 上部ツールバーに緑の「実行」アイコンがある
   - **緑の「実行」アイコンをクリック**してシナリオを実行

### pywinauto自動化のポイント

1. `os.startfile(rks_path)` でRKSファイルを開く
2. ライセンスダイアログの「OK」ボタンをクリック（auto_id不明、座標またはテキスト検索）
3. エディタウィンドウが開くのを待つ（タイトルに`.rks`が含まれる）
4. 「実行」ボタン（auto_id: `ButtonRun`）をクリック

### UI要素（スクリーンショットから確認）

| 要素 | 説明 |
|------|------|
| ライセンスダイアログ | 「RPA開発版AI付き」と「RPA開発版」の選択肢、OKボタン |
| 実行ボタン | 上部ツールバーの緑色アイコン「実行」 |
| 部分実行 | 「部分実行」ボタン |
| 一行ずつ実行 | デバッグ用 |
